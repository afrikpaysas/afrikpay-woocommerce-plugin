== DESCRIPTION ==

Choose the Afrikpay features you want – and only the features you want – to reduce purchase stress and improve your user experience.

=== How to Get Started ===
* [Sign up for Afrikpay](https://www.afrikpay.com/).
* [Install the plugin](https://wordpress.org/plugins/afrikpay-payments-for-woocommerce/) on your site. 
* Get your store approved by Afrikpay, and start selling.

__Afrikpay Payments__ removes the headaches of payments, for both consumers and merchants. Complement your checkout with a Afrikpay hosted widget located in your existing checkout which offers payment options for customers with a smooth user experience.

== Installation ==
1. Upload plugin folder to to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go WooCommerce Settings –> Payment Gateways and configure your Afrikpay Payments settings (You need to enter Afrikpay URL and the Merchant ID)

== Frequently Asked Questions ==
= Which currency does Afrikpay support? =
Afrikpay Support for the moment only FCFA (XAF)

= Where can I find Afrikpay Payments for WooCommerce documentation? =
For help setting up and configuring Afrikpay Payments for WooCommerce please refer to our [documentation](https://docs.woocommerce.com/document/afrikpay-payments/).

= Are there any specific requirements? =
* WooCommerce 3.0 or newer is required.
* PHP 5.6 or higher is required.
* A SSL Certificate is required.
